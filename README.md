С помощью приложения Draft Creator пользователи могут нарисовать изображение. Меню дает пользователям возможность открыть и отредактировать существующий файл изображения, сохранить изображение и выйти из приложения. Во время рисования меню позволяет пользователям выбирать цвет карандаша, настройку и толщину линии, тип линии, а также возможность очистки холста.
