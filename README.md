# **Draft Creator**

Простое приложение для рисования и создания эскизов. Интуитивный инструмент для быстрых набросков, заметок и творческих идей.

## Возможности

- Выбор типа линии: сплошная, пунктирная, штриховая.
- Настройка цвета пера из палитры.
- Регулировка толщины пера.
- Сохранение рисунка в галерею приложения.
- Открытие и продолжение работы с сохранёнными рисунками.
